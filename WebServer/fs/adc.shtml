    This page allows you to get continuously the ADC1 Channel 10 analog input converted value. This ADC Channel is connected to the IAR-STM32F407ZG-SK board's potentiometer. The ADC value is updated, automatically, each 1 second with the last converted ADC Channel 10 value. You could check this by changing the potentiometer position and check that the ADC value is updated, by an automatic refresh of this
