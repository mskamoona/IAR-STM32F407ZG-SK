STM32F407ZG-SK Webserver Demo
Home Page On-board LEDs control ADC Demo


    This demo shows how to use the IAR-STM32F407ZG-SK board as an embedded webserver. It is based on lwIP TCP/IP stack v1.3.2 and features three HTML pages for interacting with the IAR-STM32F407ZG-SK board.
    You will be able to control the four LEDs on the board through the browser and you will be able to get continuously, each 1 second, the ADC Channel 10 converted value displayed on the web page of your browser. The demo can be configured to use Static IP or DHCP at compile time. The address that is selected will be shown on the on-board LCD display.

* * *

    lwIP, pronounced lightweight IP, is an open source TCP/IP stack developed by Adam Dunkels at the Swedish Institute of Computer Science and is maintained now by a world wide community of developers.

lwIP features:

- IP (Internet Protocol) including packet forwarding over multiple network interfaces
- ICMP (Internet Control Message Protocol) for network maintenance and debugging
- UDP (User Datagram Protocol) including experimental UDP-lite extensions
- TCP (Transmission Control Protocol) with congestion control, RTT estimation and fast recovery/fast retransmit
- Specialized raw API for enhanced performance
- Optional Berkeley-alike socket API
- DHCP (Dynamic Host Configuration Protocol)
- PPP (Point-to-Point Protocol)
- ARP (Address Resolution Protocol) for Ethernet

For more informations you can refer to the website:  http://savannah.nongnu.org/projects/lwip/
